This is a cartridge based gaming handheld that is meant to run Pico 8. Right now, this is just a place for the bashscripts needed to get the cartridges to function correctly.

This project requires Pico 8 that can be purchased here: https://www.lexaloffle.com/pico-8.php

Dependencies:

cmake, wiring pi, git, https://github.com/juj/fbcp-ili9341.git

Licensed under GNU GPL V2. Use project at your own risk.
